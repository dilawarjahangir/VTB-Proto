## Single Page Application (SPA)

This project is configured as a Single Page Application hosted on GitHub Pages with client-side routing. All requests are processed through `index.html`.

### GitHub Pages Configuration

- **Hosting:** GitHub Pages on the `main` branch
- **Routing:** SPA-style routing with 404.html fallback
- **Jekyll:** Disabled (`.nojekyll`)

### Features

- ✅ SPA routing enabled
- ✅ Client-side navigation
- ✅ All requests processed by index.html
- ✅ Optimized for GitHub Pages deployment
